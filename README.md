## 🛠️ Tech Stack & Architecture

This project was engineered to deliver a premium agency-level experience while maintaining lightning-fast load times. It is completely framework-free, relying entirely on vanilla web technologies and advanced CSS.

* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Animation Engine:** GSAP (GreenSock) & ScrollTrigger
* **Styling Architecture:** Custom CSS Design Token System (`:root` variables)
* **Hosting:** Vercel / GitHub Pages

---

## ✨ Key Technical Features

* **Zero-Dependency Cinematic Background:** The mountain silhouette and sky gradient are rendered entirely using pure CSS (`clip-path: polygon` and `linear-gradient`), resulting in zero bandwidth cost and instant load times.
* **GSAP Scroll Choreography:** Complex scroll-triggered animations including floating dragons, parallax fog, and dynamic element reveals.
* **Custom Magnetic UI:** Engineered a custom JavaScript magnetic cursor and 3D tilt-card hover states for an immersive user experience.
* **Accessibility First (a11y):** Implemented strict `@media (prefers-reduced-motion: reduce)` fallbacks to ensure motion-sensitive users get a clean, static experience without breaking the layout.
* **Glassmorphism UI:** Advanced CSS backdrop-filters integrated seamlessly over animated backgrounds.
